# CProg 

## About the Repository
**CProg** is a collection of **C programming practice programs** created to build strong fundamentals in the C language.  
The repository is organized into **topic-wise sets**, making it easy for beginners and undergraduate students to understand, practice, and revise core C concepts.

This repo mainly focuses on **logic building**, **problem-solving**, and **hands-on coding**, which are essential for academic exams, lab work, and interviews.

---

## Topics Covered
The repository includes programs related to:

- Loops and Control Structures  
- Arrays and 2D Arrays  
- Number-based Programs  
- String Manipulation  
- Pointers and Memory Concepts  
- Structures and User-defined Data Types  
- Logical and Practice-based Problem Sets  

Each folder contains multiple C programs demonstrating practical usage of these concepts.

---

## Purpose
This repository is created to:
- Help **students** practice C programming
- Serve as a **reference** for lab assignments
- Improve **core programming logic**
- Act as a **revision guide** for exams and interviews

---

## Who Can Use This?
- Beginners learning C programming  
- Undergraduate students (B.Sc / B.Tech / BCA / Diploma)  
- Anyone revising **C language fundamentals**
